# 發財靈籤 - 樂透選號 (Caishen)

### 🀄 傳統民俗與現代技術的完美結合，為您的開獎體驗增添財氣

## 🌟 專案特色

「發財靈籤」不僅是一個樂透選號工具，更是一個充滿節慶氛圍的互動式網頁應用。

- **🧧 沉浸式視覺體驗**：
  - **3D 傾斜卡片**：隨鼠標滑動展現精美的 3D 透視效果，模擬實體開獎卡質感。
  - **金球滾動動畫**：每 80ms 高頻更新，模擬最真實的「開獎中」動感。
  - **財氣感應特效**：選號完成時觸發「大吉」蓋章、金幣掉落與碎紙花慶祝特效。
- **🎰 支援多種熱門遊戲**：
  - **大樂透**：49 選 6。
  - **威力彩**：A 區 38 選 6，B 區 8 選 1。
  - **今彩 539**：39 選 5。
  - **🛠 自定義選號**：可自由設定範圍與個數，打造專屬您的發財靈感。
- **📜 求籤互動**：通過模擬「搖晃籤筒」的儀式感，獲取對應的靈籤與祝福。
- **📱 PWA 支援**：支援離線使用與桌面安裝，隨時隨地開啟您的幸運時刻。

## 🛠️ 技術架構

本專案採用最前沿的前端技術棧開發，兼顧性能與視覺效果：

- **核心框架**：[React 19](https://react.dev/) (使用 TypeScript 進行強型別定義)
- **建構工具**：[Vite 6](https://vitejs.dev/) - 極速熱更新與優化的打包策略。
- **動畫系統**：[Framer Motion](https://www.framer.com/motion/) - 驅動流暢的組件過場與物理感知動畫。
- **音效系統**：應用 **Web Audio API** 進行程序化生成，無需加載外部大音檔即可產生豐富的鑼聲與中獎音。
- **樣式系統**：[Tailwind CSS](https://tailwindcss.com/) - 響應式佈局與現代東方美學設計。
- **離線支援**：[Vite PWA](https://vite-pwa-org.netlify.app/) - 完整的漸進式 Web 應用支持。

## 🚀 快速開始

### 環境需求

- [Node.js](https://nodejs.org/) (建議 v18.x 或以上版本)

### 本地開發步驟

1. **安裝依賴**：

   ```bash
   npm install
   ```

2. **啟動開發伺服器**：

   ```bash
   npm run dev
   ```

3. **建構生產版本**：

   ```bash
   npm run build
   ```

## 📂 專案結構簡述

- `App.tsx`: 主入口，負責全局佈局、背景裝飾與頁面切換。
- `AudioService.ts`: 程序化音頻服務。
- `components/`: 包含 `GameCard`, `FortuneRitual`, `Ball`, `HistoryPanel` 等核心組件。
- `constants.ts`: 樂透規則與靈籤詩句數據配置。
